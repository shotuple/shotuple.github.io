### 8393번
```py
N = int(input())

for i in range(N):
  N += i
print(N)
```

### 25304번 
```py
X = int(input())
N = int(input())
R = int()

for i in range(N):
  A, B = map(int, input().split())
  R += A*B
if X == R:
  print("Yes")
elif X != R:
  print("No")
```
